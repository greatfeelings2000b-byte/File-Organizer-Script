# File-Organizer-Script
This project is a Python-based automation script that organizes files into folders based on their type (extension).
## Features
- Automatically detects file types
- Creates folders if they don't exist
- Moves files into appropriate directories (Images, Documents, Audio, etc.)
- Example

Before:

Downloads/
  images.jpg
  document.pdf
  song.mp3

After:

Downloads/
  Images.jpg
  Document.pdf
  Song.mp3

## How to Run
python3 main.py

## Tech Used
- Python
- os module
- shutil module
